Comprehensive Setup Guide

Follow these steps carefully to ensure maximum safety and optimal performance for your automation matrix.

1

System Settings

Go to the System Settings tab to configure parameters as desired.

Crucial Step: You should turn off the energy saver feature in Edge (Settings -> System and Performance -> Power -> Turn Off Enable Energy Saver). This ensures the background browser runs smoothly without being throttled by the OS.

2

Telegram Remote Control

Setting up the Telegram Bot is highly recommended. It allows you to receive emergency notifications (with screenshots attached) as soon as an account hits a checkpoint.

Beyond alerts, connecting the bot helps you conveniently monitor the Tool's actual activities and control it remotely using chat commands.

3

Profile Setup

Navigate to the Profile Setup tab. You must go to the "Profile Setup" tab on the Tool and click the "Open to Login" button so the system can automatically clean up and isolate a secure environment for your login.

After logging in, close the browser and click the Identify button. The Tool will automatically fetch the Real Name and UID of the active Profile.

4

Prepare Content

Build your post and comment repository. To increase the randomness of posts and avoid being marked as Spam, users should use the Spintax structure (Content spinning) with the syntax {Content 1|Content 2|Content 3}.

To use the feature of posting or commenting with an image, you must provide at least 1 image (.jpg or .png format) to the Tool.

5

Prepare Targets

Enter the list of key Groups they want to nurture engagement with and Fanpages/Personal profiles used as a "background" for camouflaging.

For Groups, configure the Post Probability (%) and Max Visits to control how aggressively the bot interacts with each specific target.

6

Follow Balancing (Exclusive)

A specialized tool for Fanpages (Profile Pages) to clean up the "Following" list. The system will cross-check and automatically Unfollow people/pages that do not follow you back.

You can also enable the Stealth Guardian. When you check this feature, the Tool will automatically "wake up" during rest times to silently clean up the list.

7

Preview & Run Script

Return to the Dashboard. You must click the Preview Script button to let the system pre-calculate the daily task allocation matrix based on the number of Groups, Pages, and the overall health of the LIVE Profiles.

Once satisfied with the estimate, click Run Script to finalize the estimated script and activate the BotWorker.

PRO TIPS FOR SMOOTH OPERATION

  • Always use Telegram: Don't skip the bot setup. It is the best way to receive emergency notifications and issue commands securely when you are away from your PC.
  • Administrator Privileges: If the Tool fails to start or encounters an error where it cannot control the Edge browser in Headless mode, try running the Tool as an Administrator.
  • Avoid Browser Interference: To avoid interference, do not use Microsoft Edge while the Tool is running (you can use other browsers like Chrome, Firefox, etc.). If you want to watch the bot work without headless mode, use the "Show Edge" button on the dashboard.
