Important Notes (must read before downloading and installing AutoEdgeFB)

  • Operating environment: The Tool only works on Windows OS and Microsoft Edge browser. Minimum Requirements: RAM > 2GB - Storage > 1GB Recommended Windows version for VPS: Windows 10 IoT Enterprise LTSC 2021 x64 - Download Here
  • Anti-Virus: Anti-Virus Programs (Windows Defender) may mistakenly identify the Setup File (AutoEdgeFB_Setup.exe/AutoEdgeFB_Setup.zip) as a virus, so you may need to disable Anti-Virus programs when installing the Tool (More Info -> Run Anyway, or allow the Tool to run in Exclusion/Exception/Bypass mode). If the computer has a lot of important data, you should run the Tool in an Isolated or Virtualization Environment (VMWare, Docker...)
  • Run Tool as Administrator: If the Tool fails to start or encounters an error where it cannot control the Edge browser in Headless mode, try running the Tool again with Administrative privileges (Run as Administrator).
  • Optimization: Don't forget to turn off Energy Saver features in Windows and Microsoft Edge to prevent interruptions while the Tool is running.
  • Multi-account management (Multi-Profile): The Tool operates on a mechanism that creates independent Edge environments for each account (Profile). Absolutely do not open the Edge browser externally to log in. You must go to the "Profile Setup" tab on the Tool, click the "Open to Login" button so the system automatically cleans up and isolates a safe environment for you to log in.
  • Facebook Language Support: This tool only supports Facebook accounts with the language set to English or Vietnamese. To change it, from your Facebook PERSONAL Page, go to: Settings & Privacy → Language → Facebook Language → English/Vietnamese.
  • Read the Guide carefully: Before running the Tool, you must carefully read the Guide section on the Tool to understand how it operates in order to make the appropriate settings.
  • Tool Development: This is a personal project, completely free, and the author will try to spend time perfecting the Tool. When a new version is available, the Tool will be automatically updated via OTA without requiring manual action. Your contributions or constructive feedback are a great source of motivation for the Developer (please contact via Telegram for a prompt response).
